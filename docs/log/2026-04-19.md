## 本月累计经验（可提炼至 CLAUDE.md / openspec/config.yaml）

1. **Mobile/Desktop 双份 DOM + E2E `.first()` 陷阱** — `lg:hidden` / `hidden lg:block` 共存时，`.first()` 按 DOM 顺序命中隐藏副本。修复：E2E 定位必须 scope 到可见容器的 data-testid
2. **重构不能假设 UI "明显过度"** — 批次 6 把 OpponentAnalysis 两列 UTR|AI 删掉后被用户要求回滚
3. **Fly.io 256MB + 默认 JVM** — 用户反馈"以前没改时能用"，不要轻易加 `-Xmx`；OOM 从应用层收敛（MAX_OPTIONAL_POOL=18 把 C(30,8) 减到 C(18,8)）
4. **TOCTOU in JsonRepository** — `readData + writeData` 不原子，必须用 `updateData(UnaryOperator<TeamData>)`
5. **Socks5 代理会让 `npx playwright` 抛 "Protocol socks5: not supported"** — 跑 E2E 前 `unset all_proxy ALL_PROXY http_proxy ...`
6. **curl localhost 在 Windows 双栈行为** — 后端 `0.0.0.0:8080`（IPv4），Vite `[::1]:5173`（IPv6）。健康检查用明确地址：`http://127.0.0.1:8080`、`http://[::1]:5173`
7. **后端 bytecode stale** — 修改 Java 后不重启，新端点返回 `{"code":"INTERNAL_ERROR"}` 假装是 bug（实际是 `NoHandlerFoundException` 被 GlobalExceptionHandler 吞了）
